C program with Palindrome C 
// Check if a number is palindrome
int n, rev = 0, temp;
scanf("%d", &n);
temp = n;
 
while(n != 0) {
  rev = rev * 10 + n % 10;
  n /= 10;
}
 
if(temp == rev)
  printf("Palindrome");
else
  printf("Not Palindrome");

Reverses a number and compares with original. O(log n) time complexity where n is the number of digits.

C program with Fibonacci Recursion C 
// Recursive Fibonacci Series
int fib(int n) {
  if(n <= 1)
    return n;
  return fib(n-1) + fib(n-2);
}
 
int main() {
  for(int i = 0; i < 10; i++)
    printf("%d ", fib(i));
  return 0;
}

Classic recursive approach. Elegant but inefficient with O(2^n) complexity. Use memoization for optimization.

C program with Binary Search C 
// Binary search in sorted array
int binarySearch(int arr[], int n, int target) {
  int left = 0, right = n - 1;
  
  while(left <= right) {
    int mid = left + (right - left) / 2;
    
    if(arr[mid] == target)
      return mid;
    else if(arr[mid] < target)
      left = mid + 1;
    else
      right = mid - 1;
  }
  return -1;
}

Efficiently searches sorted arrays. O(log n) time complexity. Requires array to be sorted beforehand.

C program with Bubble Sort C 
// Simple sorting algorithm
void bubbleSort(int arr[], int n) {
  for(int i = 0; i < n - 1; i++) {
    for(int j = 0; j < n - i - 1; j++) {
      if(arr[j] > arr[j + 1]) {
        int temp = arr[j];
        arr[j] = arr[j + 1];
        arr[j + 1] = temp;
      }
    }
  }
}

Elementary sorting by repeatedly comparing adjacent elements. O(n²) complexity. Good for learning, not for large datasets.

C program with Factorial (Dynamic) C 
// Efficient factorial with memoization
long long factorial(int n) {
  if(n == 0 || n == 1)
    return 1;
  
  long long result = 1;
  for(int i = 2; i <= n; i++)
    result *= i;
  
  return result;
}
 
// Usage: factorial(10) → 3628800

Iterative approach avoids stack overflow. O(n) time complexity. Use long long for larger values to prevent overflow.

C program with Prime Number Check C 
// Optimized prime checker
int isPrime(int n) {
  if(n <= 1) return 0;
  if(n <= 3) return 1;
  if(n % 2 == 0 || n % 3 == 0)
    return 0;
  
  for(int i = 5; i * i <= n; i += 6)
    if(n % i == 0 || n % (i + 2) == 0)
      return 0;
  
  return 1;
}

Checks divisibility only up to √n. O(√n) complexity. 6k±1 optimization reduces operations significantly.
